# Vibe Fashion — Affordable Luxury E-commerce

A premium, conversion-optimized storefront for **@vibe_fashion_new** (Prishtinë, Kosovë) — built to
turn Instagram traffic into buyers via WhatsApp, and ready to scale into full checkout.

Built with **Next.js 14 (App Router) · TypeScript · Tailwind CSS · Framer Motion**.

---

## Quick start

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (24 routes, all products pre-rendered)
npm run start    # serve the production build
```

## ⚠️ Before you launch — edit these

Open **`src/lib/site.ts`** and replace:

| Field | Replace with |
|-------|--------------|
| `whatsappNumber` | Your real number, international format, digits only (e.g. `38344123456`) |
| `url` | Your production domain (drives canonical URLs, sitemap, OG, schema) |
| `email` | Your real contact email |
| `instagram.handle` / `url` | Already set to `vibe_fashion_new` — confirm |
| `shipping` | Confirm free-shipping threshold, fees and ETAs |

Prices, stock, sizes and copy live in **`src/lib/products.ts`** — edit freely.

## What's the design thinking?

| Decision | Why |
|----------|-----|
| **Cormorant (serif) + Montserrat (sans)** | The skill's luxury-fashion pairing — editorial headlines, clean body. |
| **Warm-neutral palette** (`cream / sand / beige / ink / gold`) | Matches your beige-black-white-gold brand. Gold (`#B6924D`) is darkened to stay WCAG-AA legible. |
| **Generous whitespace, single-column hierarchy** | The recommended high-end pattern — fewer distractions, one clear action per screen. |
| **WhatsApp as the "buy" action** | Your real sales channel today. Every CTA deep-links a pre-filled order message (item + colour + size + link). |
| **Mobile-first, thumb-friendly** | Instagram traffic is ~90% mobile. 44px+ targets, a persistent bottom CTA bar, sticky buy bar on product pages. |

## Conversion (CRO) features built in

- **Sticky "Order via WhatsApp"** — bottom bar on mobile, full-width CTA on every product, pre-filled message.
- **Scarcity** — "Only N left" badges on cards and a live pulse on product pages (driven by real `stock`).
- **Urgency** — announcement bar + a countdown "Limited drop" banner on the home page.
- **Social proof** — star ratings on every card, a testimonials section, a "verified buyers" review block.
- **Trust badges** — authenticity / shipping / returns / personal-service row under the hero.
- **Discount display** — compare-at pricing + `−%` badges, "biggest discount" sort.
- **Newsletter + exit-intent** — popup fires on exit-intent OR after 8s (once per visitor), 10%-off hook.
- **Recommendations** — "You may also like" (related) + "Recently viewed" (localStorage).
- **Wishlist** — heart toggle everywhere, badge count in navbar, dedicated page.

## SEO

- Per-page `metadata` (titles/descriptions), canonical URLs, Open Graph + Twitter cards.
- **JSON-LD**: `Store`/Organization (global), `Product` + `AggregateRating` + `Offer` (product pages),
  `BreadcrumbList`, and `FAQPage`.
- `sitemap.xml` (static + categories + products + policies) and `robots.txt` generated by Next.
- Clean, keyword-aware URLs (`/product/mk-signature-shoulder-vanilla`); semantic headings; descriptive alt text.

## Performance

- `next/image` with AVIF/WebP, responsive `sizes`, `priority` only on above-the-fold images, lazy below.
- Static pre-rendering (SSG) for all product/policy pages → fast TTFB, cacheable on a CDN.
- `font-display: swap` via `next/font` (self-hosted Google Fonts, no layout shift).
- Animations use transform/opacity only; **`prefers-reduced-motion` fully respected** (globals.css).
- Aspect-ratio boxes everywhere to keep CLS ≈ 0.

## Accessibility

- Skip-to-content link, visible gold focus rings, `aria-label`s on all icon buttons, `aria-live` on results.
- Colour is never the only signal; AA contrast on text; keyboard-operable overlays (Esc to close).

## ⭐ Biggest single upgrade: product photography

The 18 source images are **Instagram screenshots** (they include the phone status bar and IG chrome).
They work as a faithful placeholder catalog, but the #1 thing that will lift perceived value and
conversion is **clean, consistent product shots** (ideally 3:4, neutral background). Drop replacements
into `public/products/` using the same filenames and everything updates automatically.

## Scaling roadmap

1. **CMS / DB** — move `src/lib/products.ts` to Sanity/Shopify/Supabase; the data shape is already typed.
2. **Real checkout** — add Stripe/Shopify alongside WhatsApp; reuse the `Product` type and cart-ready wishlist.
3. **Live Instagram feed** — swap the static grid in `InstagramFeed.tsx` for the IG Basic Display API.
4. **Newsletter** — wire `NewsletterPopup`/`NewsletterInline` to Klaviyo/Mailchimp.
5. **Analytics** — add GA4 / Meta Pixel; `data-analytics="whatsapp-order"` hooks are already on CTAs.
6. **i18n** — add Albanian (`sq`) alongside English via `next-intl`.
